<:charter:titre_charte:> — <:charter:titre_boutons_icones:>

Les filtres |icone_horizontale et |icone_verticale transforment des liens de navigation en boutons embarquant une icône de grande taille.

Attention : ils ne sont pas appropriés pour des liens d’actions tels que \#URL_ACTION_AUTEUR, utiliser des boutons d’action pour ces derniers.

[(#INCLURE{prive/squelettes/inclure/charte/icone_verticale_horizontale.html}|dev_afficher_code{spip})]

Objet

Le 2ème paramètre permet de choisir une icône : on indique soit un type d’objet éditorial, soit une image du thème de l’espace privé.

+ Voir le code

[(#INCLURE{prive/squelettes/inclure/charte/icone_verticale_objet.html}|dev_afficher_code{spip})]

Taille

Par le biais du le 2ème paramètre on peut également préciser une taille en la suffixant.
3 tailles sont possibles : 16, 24, 32

Action

Le 3ème paramètre indique une action, son symbole est ajouté par dessus l'icône : #VALEUR
NB — del provoque l’ajout automatique de la classe .danger.

+ Voir le code

Alignement

Le 4ème paramètre permet d’ajouter une classe d’alignement : left, right ou center.
Les icônes centrées prennent une ligne entière.

+ Voir le code
