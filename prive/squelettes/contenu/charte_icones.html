Icônes

Un filtre générant des icônes à appliquer sur des liens menant vers des pages d’édition ou autre.
Il existe 2 variantes : une verticale et une horizontale.

Nb – pour des actions directes, il faut utiliser la balise #BOUTON_ACTION qui génère un formulaire sécurisé.

#BOITE_OUVRIR{'|icone_verticale'}
Syntaxe
[(#URL|icone_verticale{chaîne de langue, objet, action, alignement})]
Objet éditorial / image

Le 2ème paramètre indique le type d’objet éditorial.
Nb – Concrètement, cela correspond à une image, on peut théoriquement utiliser n’importe quelle image du thème de l’espace privé.

[(#SELF|icone_verticale{<:info_articles_2:>, article, '', left})]
[(#SELF|icone_verticale{<:texte_vider_cache:>, cache-empty-24, '', left})]
[(#SELF|icone_verticale{<:icone_activer_cookie:>, cookie-24, '', left})]

[(#SELF|icone_verticale{<:info_articles_2:>, article, '', left})] [(#SELF|icone_verticale{<:texte_vider_cache:>, cache-empty-24, '', left})] [(#SELF|icone_verticale{<:icone_activer_cookie:>, cookie-24, '', left})]
Tailles

Il existe en général 3 tailles pour les icônes des objets éditoriaux, que l’on peut éventuellement préciser : 16, 24, 32

[(#SELF|icone_verticale{<:icone_ecrire_article:>, article-16, add, left})] [(#SELF|icone_verticale{<:icone_ecrire_article:>, article-24, add, left})] [(#SELF|icone_verticale{<:icone_ecrire_article:>, article-32, add, left})]

[(#SELF|icone_verticale{<:icone_ecrire_article:>, article-16, add, left})] [(#SELF|icone_verticale{<:icone_ecrire_article:>, article-24, add, left})] [(#SELF|icone_verticale{<:icone_ecrire_article:>, article-32, add, left})]
Action

Le 3ème paramètre permet d’indiquer une action : add, edit ou del.

[(#SELF|icone_verticale{<:icone_creer_rubrique:>, rubrique, add, left})]
[(#SELF|icone_verticale{<:icone_modifier_rubrique:>, rubrique, edit, left})]
[(#SELF|icone_verticale{<:icone_supprimer_rubrique:>, rubrique, del, left})]

[(#SELF|icone_verticale{<:icone_creer_rubrique:>, rubrique, add, left})] [(#SELF|icone_verticale{<:icone_modifier_rubrique:>, rubrique, edit, left})] [(#SELF|icone_verticale{<:icone_supprimer_rubrique:>, rubrique, del, left})]
Classe

Le 4ème paramètre permet d'ajouter une classe.
left, right et center définissent ’alignement.

[(#SELF|icone_verticale{<:icone_ecrire_article:>, article, add, center})] [(#SELF|icone_verticale{<:icone_ecrire_article:>, article, add, left})] [(#SELF|icone_verticale{<:icone_ecrire_article:>, article, add, right})]
