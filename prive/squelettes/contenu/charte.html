Le menu a onglet est généré par le code [(#VAL{nomdelabarreonglet}|barre_onglets{pagecourante})]
Ce menu n'a pas l'apparrence de vrais onglets car il constitue une navigation avec changement de page. Ce menu est extensible par les plugins car la liste des pages le constituant est decrite dans paquet.xml

Des onglets simples a deux niveaux

Les onglets simples sont utilisés preferentiellement pour de la navigation ou du filtrage au sein de la même page. Ils sont utilisés avec des liens ajax pour être plus réactifs, ce qui est attendu par les utilisateurs confrontés à un onglet.
Il n'y a pas de filtre pour les construire car leur structure HTML est très simple.
